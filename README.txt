How to run
----------
1. Download the notebook.
	If the downloaded notebook is DIP_Abstraction_and_uniqueness.ipynb Please run it only on google colab only. Before running it, please upload the image 'leaf.jpg'.
	This notebook will run only if the image is leaf.jpg. If the user wants to run on a different image, the user has to change line number 32 image=cv.imread('imagename')
	(Download the notebook that you want to run for eg if you want to run the entire project run final_project.ipynb)

2. Download the image set. 

2. Include the all the images in file where the notebook is run. 

3. Run the notebook.
